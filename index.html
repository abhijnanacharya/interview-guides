Interval Problems in Competitive Programming

A Comprehensive Guide to Sorting and Priority Queue Approaches

Content Overview

This report covers the essential interval-based problems in competitive programming that require either sorting or priority queue approaches. These problems are fundamental in algorithm design and appear frequently in coding interviews and contests.

Problem Categories

SORTING APPROACH
1. Interval Merging
Problem: Given a collection of intervals, merge overlapping intervals.
Strategy: Sort by start time, then iterate and merge overlapping intervals.
vector<vector<int>> mergeIntervals(vector<vector<int>>& intervals) { sort(intervals.begin(), intervals.end()); vector<vector<int>> merged; merged.push_back(intervals[0]); for (int i = 1; i < intervals.size(); i++) { if (intervals[i][0] <= merged.back()[1]) { merged.back()[1] = max(merged.back()[1], intervals[i][1]); } else { merged.push_back(intervals[i]); } } return merged; }
Time: O(n log n)     Space: O(1)
SORTING APPROACH
2. Activity Selection
Problem: Select maximum number of non-overlapping intervals.
Strategy: Sort by end time, greedily select intervals that don't overlap.
int maxNonOverlapping(vector<vector<int>>& intervals) { sort(intervals.begin(), intervals.end(), [](const vector<int>& a, const vector<int>& b) { return a[1] < b[1]; }); int count = 1; int lastEnd = intervals[0][1]; for (int i = 1; i < intervals.size(); i++) { if (intervals[i][0] >= lastEnd) { count++; lastEnd = intervals[i][1]; } } return count; }
Time: O(n log n)     Space: O(1)
PRIORITY QUEUE
3. Meeting Rooms
Problem: Find minimum number of meeting rooms needed.
Strategy: Sort by start time, use min-heap to track room end times.
int minMeetingRooms(vector<vector<int>>& intervals) { sort(intervals.begin(), intervals.end()); priority_queue<int, vector<int>, greater<int>> minHeap; for (auto& interval : intervals) { if (!minHeap.empty() && minHeap.top() <= interval[0]) { minHeap.pop(); } minHeap.push(interval[1]); } return minHeap.size(); }
Time: O(n log n)     Space: O(n)
EVENT-BASED
4. Max Overlapping
Problem: Find maximum number of intervals that overlap at any point.
Strategy: Convert intervals to start/end events, sort and sweep.
int maxOverlapping(vector<vector<int>>& intervals) { vector<pair<int, int>> events; for (auto& interval : intervals) { events.push_back({interval[0], 1}); // Start event events.push_back({interval[1], -1}); // End event } sort(events.begin(), events.end()); int maxOverlap = 0, currentOverlap = 0; for (auto& event : events) { currentOverlap += event.second; maxOverlap = max(maxOverlap, currentOverlap); } return maxOverlap; }
Time: O(n log n)     Space: O(n)
PRIORITY QUEUE
5. Skyline Problem
Problem: Given buildings with heights, find the skyline silhouette.
Strategy: Use events for building start/end, maintain active heights.
vector<vector<int>> getSkyline(vector<vector<int>>& buildings) { vector<pair<int, int>> events; for (auto& building : buildings) { events.push_back({building[0], -building[2]}); events.push_back({building[1], building[2]}); } sort(events.begin(), events.end()); vector<vector<int>> result; multiset<int> heights = {0}; for (auto& event : events) { int pos = event.first; int heightChange = event.second; if (heightChange < 0) { heights.insert(-heightChange); } else { heights.erase(heights.find(heightChange)); } int maxHeight = *heights.rbegin(); if (result.empty() || result.back()[1] != maxHeight) { result.push_back({pos, maxHeight}); } } return result; }
Time: O(n log n)     Space: O(n)
SORTING APPROACH
6. Interval Insertion
Problem: Insert a new interval into sorted non-overlapping intervals.
Strategy: Find insertion position, merge overlapping intervals.
vector<vector<int>> insertInterval(vector<vector<int>>& intervals, vector<int>& newInterval) { vector<vector<int>> result; int i = 0; // Add intervals before new interval while (i < intervals.size() && intervals[i][1] < newInterval[0]) { result.push_back(intervals[i]); i++; } // Merge overlapping intervals while (i < intervals.size() && intervals[i][0] <= newInterval[1]) { newInterval[0] = min(newInterval[0], intervals[i][0]); newInterval[1] = max(newInterval[1], intervals[i][1]); i++; } result.push_back(newInterval); // Add remaining intervals while (i < intervals.size()) { result.push_back(intervals[i]); i++; } return result; }
Time: O(n)     Space: O(1)

ðŸŽ¯ Decision Framework

Sort by START time

When: Merging intervals, processing chronologically

Examples: Merge Intervals, Insert Interval

Sort by END time

When: Greedy selection problems

Examples: Activity Selection, Point Coverage

Priority Queue

When: Dynamic tracking of max/min

Examples: Meeting Rooms, CPU Scheduling

Event-based

When: Counting overlaps, state changes

Examples: Max Overlapping, Skyline
